# Degraded/Paywalled Originally FOSS Tools
 * [Coalitioninc](https://control.coalitioninc.com/)
	 * Seems to be restricted to policy holders now... 
 * [eReKon](https://github.com/rubenkharel/eReKon)
	 * Uncertain if this is usable as-is
	 * README says `unstable-rn ongoing-rebuild in new stack`
 * [Findomain](https://github.com/Findomain/Findomain) *
	 * OSS, but not free
 * [envizon](https://github.com/evait-security/envizon)
	 * End of life/archived
 * [hakluke's open source asm with spiderfoot](https://hakluke.com/open-source-asm-spiderfoot)*
	 * No longer managed in hakluke's repo; it was sold to Intel471, but the [repo is still live](https://github.com/smicallef/spiderfoot)
 * [intrigue-core](https://github.com/intrigueio/intrigue-core)
	 * Now owned and managed by Mandiant under a Limited Open Source agreement, and is no longer maintained on GitHub
 * [natlas](https://github.com/natlas/natlas) *
	 * Last commit is June 2021
 * [SonarSearch](https://github.com/Cgboal/SonarSearch)
	 * Requires access to the [Rapid7 Project Sonar](https://www.rapid7.com/research/project-sonar/) datasets, which are behind a registration wall.
